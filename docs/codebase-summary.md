# Codebase Summary

## Overview

Static landing page for landing page design service targeting Vietnamese market.

## Files

| File | Lines | Purpose |
|------|-------|---------|
| `index.html` | ~700 | Complete landing page |
| `css/styles.css` | ~600 | Design system, components |
| `css/responsive.css` | ~200 | Mobile breakpoints |
| `css/animations.css` | ~200 | Scroll animations |
| `js/main.js` | ~150 | Core functionality |
| `js/animations.js` | ~150 | Scroll reveals |

## Sections

1. **Hero** - USP headline, CTAs, stats counter
2. **Values** - 5 value propositions with icons
3. **Services** - 7 service cards
4. **Why Choose Us** - 5 trust factors
5. **Portfolio** - 6 project showcases
6. **Process** - 6-step workflow
7. **Pricing** - 3 tier cards
8. **Testimonials** - 3 customer reviews
9. **FAQ** - 6 accordion items
10. **Contact** - Form + social links
